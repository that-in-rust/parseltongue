# Parseltongue: Compare Java vs Rust Performance
> "Speaking both languages fluently" 🐍

## Design System

1. Typography
   ```css
   /* Primary: JetBrains Mono */
   --font-code: 'JetBrains Mono', monospace;
   /* Secondary: Inter */
   --font-ui: 'Inter', -apple-system, sans-serif;
   ```

2. Color Palette
   ```css
   /* Brand Colors */
   --java: #b07219;     /* Java warm */
   --rust: #dea584;     /* Rust earthy */
   
   /* UI Colors */
   --surface: #0d1117;  /* Dark mode background */
   --card: #161b22;     /* Card background */
   --accent: #238636;   /* Success green */
   --text: #e6edf3;     /* Primary text */
   --dim: #7d8590;      /* Secondary text */
   ```

3. Layout Components
   ```
   ┌─ Dark Mode Interface ───────┐
   │                            │
   │  🐍 Parseltongue          │
   │  <Inter Medium>           │
   │                           │
   │  ⚡ [Start Analysis]      │
   │  <Monospace + Gradient>   │
   │                           │
   └───────────────────────────┘
   ```

4. Progress Display
   ```
   ┌─ Terminal-style Card ──────┐
   │ $ analyzing react repo...  │
   │                           │
   │ ☕ Java Backend           │
   │ ▓▓▓▓▓▓░░░░ 66.7%        │
   │ → src/components/App.js   │
   │ @ 127 files/sec          │
   │                           │
   │ 🦀 Rust Backend          │
   │ Queued...                │
   └───────────────────────────┘
   ```

5. Results View
   ```
   ┌─ GitHub-style Card ────────┐
   │                           │
   │   Performance Battle      │
   │   ──────────────────     │
   │                           │
   │   Java     vs    Rust    │
   │   1.2s          0.8s     │
   │   1205 files   1205     │
   │                           │
   │   🏆 33% faster!         │
   │                           │
   └───────────────────────────┘
   ```

## Interaction States
```css
/* Button States */
.button {
  --hover: #2ea043;
  --active: #238636;
  --disabled: #238636;
}

/* Progress States */
.progress {
  --java-gradient: linear-gradient(90deg, #b07219, #d18f33);
  --rust-gradient: linear-gradient(90deg, #dea584, #e6b89c);
}

/* Card Hover */
.card {
  --hover-border: #30363d;
  --hover-shadow: 0 8px 24px rgba(140, 149, 159, 0.2);
}
```

## Animations
1. Progress Bar
   - Smooth width transitions
   - Subtle gradient movement
   - Pulsing when paused

2. Status Changes
   - Fade between states
   - Smooth height adjustments
   - Loading spinners

3. Results Reveal
   - Card flip animation
   - Number counting effect
   - Winner highlight pulse
