## Interaction States
```css
/* Button States */
.button {
  --hover: #2ea043;
  --active: #238636;
  --disabled: #238636;
}

/* Progress States */
.progress {
  --java-gradient: linear-gradient(90deg, #b07219, #d18f33);
  --rust-gradient: linear-gradient(90deg, #dea584, #e6b89c);
}

/* Card Hover */
.card {
  --hover-border: #30363d;
  --hover-shadow: 0 8px 24px rgba(140, 149, 159, 0.2);
}
```

## Animations
1. Progress Bar
   - Smooth width transitions
   - Subtle gradient movement
   - Pulsing when paused

2. Status Changes
   - Fade between states
   - Smooth height adjustments
   - Loading spinners

3. Results Reveal
   - Card flip animation
   - Number counting effect
   - Winner highlight pulse
