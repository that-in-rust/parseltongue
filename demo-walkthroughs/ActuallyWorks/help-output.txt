Ultra-minimalist CLI toolkit for code analysis and modification

Usage: parseltongue [COMMAND]

Commands:
  pt01-folder-to-cozodb-streamer        Tool 1: Stream folder contents to CozoDB with ISGL1 keys
  pt02-level00                          Tool 2a: Export pure edge list (Level 0 - ~2-5K tokens)
  pt02-level01                          Tool 2b: Export entities with ISG + temporal (Level 1 - ~30K tokens) [RECOMMENDED]
  pt02-level02                          Tool 2c: Export entities with type system (Level 2 - ~60K tokens)
  pt03-llm-to-cozodb-writer             Tool 3: Write LLM-proposed changes to temporal state
  pt04-syntax-preflight-validator       Tool 4: Validate syntax of proposed changes
  pt05-llm-cozodb-to-diff-writer        Tool 5: Generate CodeDiff.json from temporal state
  pt06-cozodb-make-future-code-current  Tool 6: Make future code current and reset temporal state
  help                                  Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
